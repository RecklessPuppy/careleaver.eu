# Prompt 03: Build V1

Use this prompt when it is time to edit the public website.

```text
We are working on careleaver.eu.

Please build the next small V1 improvement directly in index.html.

Before editing:
- read AGENTS.md, PROJECT_BRIEF.md, STATE.md, SOURCE_POLICY.md, CONTENT_SAFETY.md, and OPERATING_MODEL.md
- inspect git status
- inspect index.html
- confirm whether the change touches source-sensitive content

Rules:
- keep the site static and simple
- preserve CNAME and the custom domain careleaver.eu
- keep public content in German
- use Austria-first, Wien-first scope
- do not invent contact details
- source-link and date legal, health, benefits, emergency, housing, and contact claims
- make a small focused change
- commit and push to main when done

After editing, summarize what changed in beginner-friendly language.
```
