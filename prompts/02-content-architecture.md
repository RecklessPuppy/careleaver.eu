# Prompt 02: Content Architecture

Use this prompt to plan the public content structure after a research pass.

```text
We are working on careleaver.eu.

Please create a content architecture for the Austria/Wien MVP.

Read:
- AGENTS.md
- PROJECT_BRIEF.md
- ROADMAP.md
- SOURCE_POLICY.md
- CONTENT_SAFETY.md
- STATE.md
- research/source-log.md
- research/research-synthesis.md
- research/open-questions.md
- index.html

Deliver:
1. A proposed Wien MVP page structure.
2. Which existing sections should stay, change, or wait.
3. Which claims need visible source links or dates.
4. A short list of the safest first public content edits.

Do not implement a redesign yet unless I explicitly ask.
Keep public content in German.
```
